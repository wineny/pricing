# 맞춤형 워크샵 견적 계산기

2일차/4일차 프로그램의 견적을 자동으로 계산하는 시스템입니다.

## 📁 파일 구성

- **`pricing_calculator.py`**: Python 기반 CLI 계산기
- **`index.html`**: 웹 기반 대화형 계산기
- **`초안.md`**: 계산 로직 문서

## 🚀 사용 방법

### 1. Python CLI 계산기

```bash
# 실행
python3 pricing_calculator.py

# 메뉴
1. 견적 계산 (세일즈 가격)
2. 견적 계산 (원가 기반)
3. 전체 가격표 보기 (세일즈)
4. 전체 가격표 보기 (원가)
5. 프로그램 비교
0. 종료
```

### 2. 웹 계산기

```bash
# 브라우저로 열기
open index.html

# 또는 로컬 서버 실행
python3 -m http.server 8000
# http://localhost:8000 접속
```

## 💰 가격 구조

### 세일즈 최적화 가격 (권장)

| 구분 | 10-20명 | 21-30명 |
|---|---:|---:|
| **2일차** | 1,000,000원/명 | 900,000원/명 |
| **4일차** | 1,400,000원/명 | 1,200,000원/명 |

### 원가 기반 계산

#### 2일차 프로그램
- 강의료: 8시간 × 750,000원 = 6,000,000원
- 발표/진행료: 5시간 × 300,000원 = 1,500,000원
- 개별 코칭: N명 × 1시간 × 300,000원
- 보조강사: 13시간 × 300,000원 × A(N)명

**총액 = 7,500,000 + (300,000 × N) + (3,900,000 × A(N))**

#### 4일차 프로그램
- 강의료: 11시간 × 750,000원 = 8,250,000원
- 발표/진행료: 6시간 × 300,000원 = 1,800,000원
- 상주 코칭료: 11시간 × 300,000원 = 3,300,000원
- 보조강사: 28시간 × 300,000원 × A(N)명

**총액 = 13,350,000 + (8,400,000 × A(N))**

### 보조강사 자동 배정

```
A(N) = { 0명  (N ≤ 10)
       { 1명  (11 ≤ N ≤ 20)
       { 2명  (21 ≤ N ≤ 30)
```

## 🎯 핵심 특징

### 1. 구간별 단일 단가제
- 11명 vs 10명 같은 '단 1명 차이' 스트레스 제거
- 예산 예측성 향상
- CFO 설명 용이

### 2. 규모의 경제
- 20명 → 30명: 인당 10-14% 절감
- 대규모 교육 시 비용 효율성 증대

### 3. 자동 보조강사 배정
- 인원에 따라 자동 계산
- 품질 유지 보장

## 💼 세일즈 활용 팁

### 의사결정 플로우
```
Q1. 교육 기간은?
    → 2일? → 1,000,000원 or 900,000원/명
    → 4일? → 1,400,000원 or 1,200,000원/명

Q2. 참여 인원은?
    → 10-20명? → Tier 1 단가
    → 21-30명? → Tier 2 단가 (10-14% 할인)
```

### 설득 포인트
1. **예산 예측성**: 인원 변동에도 구간 내 고정 단가
2. **규모의 경제**: 대규모 교육 시 절감 효과
3. **투명한 가격**: 보조강사 비용 포함된 올인원 가격

### 경쟁 우위
- 타사: 보조강사 별도 과금 → 실제 20-30% 추가 비용
- 당사: 보조강사 포함 가격 → 예산 초과 위험 없음
