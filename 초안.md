# 맞춤형 워크샵 견적 계산

2일차, 4일차 프로그램 중 하나 선택, 인원 수에 따라서 보조 강사 수 혹은 코칭료가 달라짐. 

# 2일차 계산 로직(정리본)

## 전제(2일차)

* 공통 강의 시간: **8시간** → **강의 단가 75만원/시간**
* 공통 발표(진행) 시간: **5시간** → **코칭 단가 30만원/시간**
* 인원별 추가 코칭: **N명 × 1시간** → **30만원/시간**
* 보조강사(11~20명: 1명 / 21~30명: 2명)

  * **보조 단가 30만원/시간**,
  * **투입 구간:** 공통 강의(8h) + 공통 발표(5h) = **총 13시간**에 대해 붙음
  * (개별 인당 코칭에는 붙지 않는다고 가정)

> 즉, **최소 진행 시간(13h)**은 “메인 강사”가 수행하고, 인원이 많아지면 **보조가 동일 13시간만큼** 추가로 붙어 **시간×보조 수**만큼 비용이 더해진다고 보시면 됩니다.

---

## 공식

### 1) 최소 진행 비용(최소 10명 기준)

* **LectureRevenue(강의)** = `8h × 750,000` = **6,000,000원**
* **PresentationRevenue(발표/진행)** = `5h × 300,000` = **1,500,000원**
* **기본(최소) 진행 비용** = **6,000,000 + 1,500,000 = 7,500,000원**

### 2) 인당 단가(개별 코칭)

* **Per-Person Coaching** = `N × 1h × 300,000` = **300,000 × N 원**

### 3) 보조강사 추가비(11명 이상일 때)

* **보조 투입 시간** = `8h + 5h = 13h`
* **보조 수 A(N)** = `1 (11~20명)`, `2 (21~30명)`
* **Assistant Cost** = `13h × 300,000 × A(N)` = **3,900,000 × A(N) 원**

### 4) 총액

[
\textbf{Total(2일차)} = 7{,}500{,}000 ;+; 300{,}000 \times N ;+; 3{,}900{,}000 \times A(N)
]

* (A(N)=0) if (N \le 10)
* (A(N)=1) if (11 \le N \le 20)
* (A(N)=2) if (21 \le N \le 30)

좋아요—4일차는 “상주형”이라 훨씬 단순합니다.
핵심은 **기본(메인 강사 단독) + 보조강사(인원 구간별) 추가** 구조예요.

# 4일차 계산 로직

## 입력

* 인원 `N`
* 보조강사 수 `A(N)`

  * `N ≤ 10` → `A = 0`
  * `11 ≤ N ≤ 20` → `A = 1`
  * `21 ≤ N ≤ 30` → `A = 2`

## 시간 & 단가

* 공통 강의: **11h × 750,000**
* 공통 발표(진행): **6h × 300,000**
* 코칭(상주): **메인 11h × 300,000**  (10명 이하는 메인만, 11명↑부터 보조도 같은 시간 상주)
* 보조강사 단가: **시간당 300,000**
  → “모든 세션(강의 11h + 발표 6h + 코칭 11h)”에 동일하게 붙음

## 공식

* **기본(메인만)**
  `Base = (11×750,000) + (6×300,000) + (11×300,000)`
  `= 8,250,000 + 1,800,000 + 3,300,000 = 13,350,000`
* **보조강사 추가(인원 구간별)**
  `AssistantAdd = A(N) × 300,000 × (11 + 6 + 11)`
  `= A(N) × 300,000 × 28 = A(N) × 8,400,000`
* **총액(4일차)**
  `Total = 13,350,000 + 8,400,000 × A(N)`

## 예시 합계

| 인원 N | A(N) |             총액 |
| ---: | ---: | -------------: |
|   10 |    0 | **13,350,000** |
|   15 |    1 | **21,750,000** |
|   20 |    1 | **21,750,000** |
|   25 |    2 | **30,150,000** |
|   30 |    2 | **30,150,000** |

## 세부 항목별(참고)

* 메인 고정: 강의 8,250,000 + 발표 1,800,000 + 코칭 3,300,000 = **13,350,000**
* 보조 1명 추가 시: **+ 8,400,000**
* 보조 2명 추가 시: **+ 16,800,000**
